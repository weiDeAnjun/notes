# 高等数学



**导数的逆运算对应的是不定积分，而非定积分**。具体可以从以下角度拆解：

1. 从 “运算结果的形式” 看逆运算

- **导数运算**：输入一个函数*F*(*x*)，输出它的导函数*f*(*x*)=*F*′(*x*)。
    例如：*F*(*x*)=*x*2 → 导数运算后得到*f*(*x*)=2*x*。

- **不定积分运算**：输入一个函数*f*(*x*)，输出 “所有导函数为*f*(*x*)的函数”，即∫*f*(*x*)*d**x*=*F*(*x*)+*C*（*C*为任意常数）。
    例如：对*f*(*x*)=2*x*做不定积分，得到∫2*x**d**x*=*x*2+*C*，恰好包含了原函数*F*(*x*)=*x*2（当*C*=0时），以及所有与它相差常数的函数（如*x*2+1、*x*2−5等）。

    这说明：**不定积分是导数的 “逆运算”**—— 它能把导数运算 “丢失的常数项” 以 “+*C*” 的形式补回来，最终还原出所有可能的原函数。

2. 定积分为什么不是导数的逆运算？

定积分∫*a**b**f*(*x*)*d**x*的结果是一个**数值**（表示曲边梯形的面积），而导数的结果是一个**函数**。两者的 “运算对象” 和 “输出形式” 完全不同：

- 导数：函数 → 函数（导函数）；
- 定积分：函数 → 数值（面积）。

因此，定积分与导数不是直接的逆运算关系。

但定积分与原函数存在间接联系（牛顿 - 莱布尼茨公式）：若*F*′(*x*)=*f*(*x*)，则∫*a**b**f*(*x*)*d**x*=*F*(*b*)−*F*(*a*)。这个公式说明：**定积分可以通过原函数计算，但它本身不是导数的逆运算**。

**边缘分布仍是正态分布**

若 (*X*,*Y*)∼*N*(*μ*1,*μ*2;*σ*12,*σ*22;*ρ*)，则：

- *X*∼*N*(*μ*1,*σ*12)（仅依赖 *μ*1,*σ*12，与 *ρ* 无关）；
- *Y*∼*N*(*μ*2,*σ*22)（仅依赖 *μ*2,*σ*22，与 *ρ* 无关）。

**注意**：反之不成立 —— 若 *X* 和 *Y* 各自服从一维正态分布，(*X*,*Y*) 不一定服从二维正态分布（需额外满足 “联合密度符合二维正态形式”）。

**条件分布仍是正态分布**

给定一个变量的取值后，另一个变量的条件分布仍为一维正态分布：

- 给定 *Y*=*y* 时，*X* 的条件分布：*X*∣*Y*=*y*∼*N*(*μ*1+*ρ*⋅*σ*2*σ*1(*y*−*μ*2),*σ*12(1−*ρ*2))；
- 给定 *X*=*x* 时，*Y* 的条件分布：*Y*∣*X*=*x*∼*N*(*μ*2+*ρ*⋅*σ*1*σ*2(*x*−*μ*1),*σ*22(1−*ρ*2))。

**关键观察**：

- 条件期望是 “给定变量的线性函数”（如 *E*[*X*∣*Y*=*y*] 是 *y* 的一次函数）；
- 条件方差仅与 *σ*12,*ρ* 有关（与 *y* 无关），体现了 “方差齐性”。

3. 独立性与不相关性等价

这是二维正态分布**最独特的性质之一**（对其他分布不成立）：
若 (*X*,*Y*)∼*N*(*μ*1​,*μ*2​;*σ*12​,*σ*22​;*ρ*)，则：*X* 与 *Y* 相互独立 ⟺*ρ*=0（即 *X* 与 *Y* 线性不相关）。

**原理**：当 *ρ*=0 时，联合密度函数可拆分为 *X* 和 *Y* 的边缘密度函数的乘积（*f*(*x*,*y*)=*f**X*(*x*)⋅*f**Y*(*y*)），满足独立的定义。

4. 线性变换不变性

若 (*X*,*Y*) 服从二维正态分布，对其进行**线性变换**（如 *U*=*a**X*+*bY*+*c*，*V*=*d**X*+*e**Y*+*f*，其中 *a*,*b*,*d*,*e* 不全为 0），则新的二维变量 (*U*,*V*) 仍服从二维正态分布（或退化的一维正态分布，当变换矩阵秩为 1 时）。

这一性质是二维正态分布在统计分析（如线性回归、主成分分析）中广泛应用的核心原因。

四、特殊情况：标准二维正态分布

当所有参数取简化值时，得到**标准二维正态分布**，记为 (*X*,*Y*)∼*N*(0,0;1,1;*ρ*)，其联合密度函数简化为：

*f*(*x*,*y*)=2*π*1−*ρ*21exp{−2(1−*ρ*2)1(*x*2−2*ρ**x**y*+*y*2)}

此时，*X*∼*N*(0,1)、*Y*∼*N*(0,1)（标准一维正态），相关系数仍为 *ρ*。

五、一句话总结

二维正态分布是描述两个连续变量联合分布的 “基准模型”，由**两个边缘均值、两个边缘方差、一个相关系数**共 5 个参数完全决定，核心性质包括 “边缘 / 条件分布仍为正态”“独立性与不相关性等价”“线性变换不变”，是概率统计、金融、工程等领域建模的重要工具。
